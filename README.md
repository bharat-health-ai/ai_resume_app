# AI Resume Builder App

**Description:**  
AI-powered Resume & Cover Letter Builder. Generate resumes in Hindi, download PDF, and pay for premium features via Razorpay.

**Features:**  
- AI-generated resumes & cover letters  
- PDF download  
- Razorpay payment integration  
- Flutter mobile frontend  
- Python FastAPI backend  
- Google AI Studio integration (placeholder)
